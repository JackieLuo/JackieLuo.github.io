语法：

  • enumerate(an_iter)
  • enumerate(an_iter, start)

示例：

代码 输出
print enumerate([‘a’, ‘b’, ‘c’, ‘d’]) enumerate object
print list(enumerate([‘a’, ‘b’, ‘c’, ‘d’])) [(0, ‘a’), (1, ‘b’), (2, ‘c’), (3, ‘d’)]
print list(enumerate([‘a’, ‘b’, ‘c’, ‘d’], 5)) [(5, ‘a’), (6, ‘b’), (7, ‘c’), (8, ‘d’)]
